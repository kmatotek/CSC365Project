\begin{block}{Introduction}
As LLMs like ChatGPT grow more capable, distinguishing AI-generated text from human writing becomes increasingly difficult --- yet increasingly important for academic integrity, journalism, and trust online.

We ask: \textbf{can a simple, interpretable model do the job?} Rather than using a heavyweight neural detector, we train a logistic regression on TF-IDF features --- a model whose decisions can be fully audited and explained.

\vspace{0.3em}
\textbf{Research Questions:}
\begin{itemize}
  \item Can lightweight models reliably detect AI-generated text?
  \item Which linguistic patterns most distinguish ChatGPT from humans?
  \item Does detection hold across different topic domains?
\end{itemize}
\end{block}

\begin{block}{Methodology}

\textbf{Dataset --- HC3:}
\begin{itemize}
  \item Human and ChatGPT answers to the same questions
  \item Domains: Reddit, StackExchange, medicine, finance
  \item Split \textbf{by question group} to prevent leakage
\end{itemize}

\vspace{0.3em}
\textbf{Model Pipeline:}
\begin{itemize}
  \item TF-IDF (50k features, unigrams + bigrams)
  \item Logistic Regression with balanced class weights
  \item Fully interpretable coefficients
\end{itemize}

\end{block}

\vspace{-0.3cm}

\begin{block}{Findings}

\begin{itemize}
  \item \textbf{$>$95\% accuracy} and ROC-AUC $>$ 0.98 on held-out test data
  \item ChatGPT text is \textbf{stylistically uniform}; human text is diverse
  \item A handful of formulaic phrases are near-\textbf{exclusive signals} of LLM output
  \item Performance generalizes strongly across all six source domains
\end{itemize}

\vspace{0.5em}
\textbf{Takeaways:}
\begin{itemize}
  \item Lightweight models \textbf{match neural detectors} at a fraction of the cost
  \item Interpretability lets us \textbf{explain every decision} --- a key advantage for real-world use
  \item As LLMs improve, lexical patterns will weaken; future detectors will need stylometric and syntactic features
\end{itemize}

\end{block}
