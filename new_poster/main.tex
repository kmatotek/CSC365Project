\begin{block}{Introduction}
The rapid rise of large language models (LLMs) such as ChatGPT has made it increasingly difficult to distinguish between human-written and AI-generated text.

This project explores whether \textbf{simple, interpretable models} using linguistic features can reliably detect AI-generated content.

\textbf{Research Questions:}
\begin{itemize}
\item Can lightweight models distinguish AI vs human text?
\item What linguistic patterns differentiate the two?
\item How well does performance generalize across sources?
\end{itemize}
\end{block}

\vspace{-0.5cm}

\begin{block}{Methodology}

\textbf{Dataset:}
\begin{itemize}
\item HC3 dataset (Human vs ChatGPT responses)
\item Multiple domains: Reddit, StackExchange, etc.
\end{itemize}

\textbf{Model:}
\begin{itemize}
\item TF-IDF vectorization
\item Logistic Regression classifier
\item Interpretable linear weights
\end{itemize}

\textbf{Evaluation:}
\begin{itemize}
\item Accuracy
\item ROC-AUC
\item Source-wise generalization
\end{itemize}

\end{block}

\begin{block}{Findings}

\textbf{Key Observations:}
\begin{itemize}
\item AI-generated text is highly consistent in style
\item Certain phrases and structures strongly signal LLM output
\item Human text exhibits greater variability
\end{itemize}

\textbf{Implications:}
\begin{itemize}
\item Lightweight models remain competitive for detection
\item Interpretability is a major advantage over deep models
\item Detection may become harder as LLMs improve
\end{itemize}

\end{block}

\begin{block}{Future Work}
\begin{itemize}
\item Evaluate robustness on newer LLMs (GPT-4+, Claude, etc.)
\item Explore adversarial paraphrasing attacks
\item Incorporate syntactic and semantic features
\item Compare against deep neural classifiers
\end{itemize}
\end{block}
