\begin{block}{Introduction}
As LLMs like ChatGPT become more capable, it’s getting harder to tell AI-generated text apart from human writing. This matters for academic integrity, journalism, and trust online.

We ask: \textbf{can a simple, interpretable model do the job?} Instead of a complex neural detector, we use logistic regression with TF-IDF features.

\vspace{0.3em}
\textbf{Research Questions:}
\begin{itemize}
  \item Can lightweight models reliably detect AI-generated text?
  \item Which linguistic patterns most distinguish ChatGPT from humans?
  \item Does detection hold across different topic domains?
\end{itemize}
\end{block}

\begin{block}{Methodology}

\textbf{Dataset (HC3):}
\begin{itemize}
  \item Human vs. ChatGPT answers to the same questions
  \item Domains: Reddit (ELI5), open QA, Wikipedia (CS), medicine, finance
  \item Split by question group to prevent leakage
\end{itemize}

\vspace{0.3em}
\textbf{Model:}
\begin{itemize}
  \item TF-IDF (50k features, unigrams + bigrams)
  \item Logistic Regression (balanced classes)
  \item Train/val/test split: 75\% / 10\% / 15\% 
\end{itemize}

\end{block}

\vspace{-0.3cm}

\begin{block}{Findings}

\begin{itemize}
  \item \textbf{$>$95\% accuracy} on held-out test data
  \item ChatGPT text is \textbf{stylistically uniform}, while human text is more diverse
  \item ChatGPT features skew toward \textbf{common, generic terms}, while human features reflect more personal and informal language
  \item Performance generalizes across all five domains
\end{itemize}

\vspace{0.5em}
\textbf{Takeaways:}
\begin{itemize}
  \item Lightweight models achieve strong performance at a fraction of the cost of neural detectors
  \item Interpretability allows us to \textbf{explain model decisions}, which is valuable in real-world settings
  \item As LLMs improve, lexical signals may weaken, requiring more advanced stylistic features
\end{itemize}

\end{block}
